## Repository maturity

[Muze](https://muze.nl) uses two GitHub organizations and two npm namespaces:

- `muze-nl` / `@muze-nl` is for libraries that are close to production-ready.
- `muze-labs` / `@muze-labs` is for experimental or incubating projects.

Moving a project from Muze Labs to Muze NL is a graduation step.
